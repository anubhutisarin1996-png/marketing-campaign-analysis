# Marketing Campaign Analysis

## Project Overview
This project focuses on analyzing marketing campaign performance to identify customer engagement trends, campaign effectiveness, and ROI optimization opportunities.

---

## Tools Used
- Excel
- SQL
- Power BI

---

## Project Workflow
1. Data Cleaning
   - Checked missing values
   - Removed duplicate records

2. Excel Analysis
   - Created pivot tables
   - Analyzed campaign performance

3. SQL Analysis
   - Performed KPI calculations
   - Customer segmentation
   - Revenue and conversion analysis

4. Power BI Dashboard
   - Built interactive dashboards
   - Visualized marketing KPIs

---

## Key KPIs
- Conversion Rate
- Customer Acquisition Cost
- ROI
- Campaign Revenue
- Engagement Metrics

---

## Key Insights
- Identified top-performing marketing channels
- Found customer segments with highest conversions
- Analyzed campaign ROI trends
